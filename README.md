# Enhanced Image Super-Resolution

A deep learning application that uses a **Residual ANN** to upscale low-resolution images. Features a simple web interface built with Streamlit.

### Tech Stack
**Core Tech:** Python | TensorFlow/Keras | Streamlit | NumPy
